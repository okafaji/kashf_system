## ✅ تم إصلاح مشكلة حساب الإيفاد - خارج البلد

### 📋 ما تم تصحيحه:

1. ✅ **تحويل البيانات إلى أرقام**
   - تحويل `daily_rate` من string إلى number في `create.blade.php`
   - يضمن الحساب الصحيح

2. ✅ **دالة بسيطة لتحديث الأسعار**
   - دالة `updateMissionRateFromLevel($row)` في `payroll_manager.js`
   - تبحث عن الإيفاد + المستوى الوظيفي في البيانات المحملة
   - تحدّث السعر اليومي تلقائياً

3. ✅ **تحسين معالج تغيير الوجهة**
   - عند اختيار خارج القطر: إظهار حقل المستوى الوظيفي
   - عند اختيار مدينة: إظهار حقل العنوان الوظيفي وتعيين سعرها
   - تصفير السعر اليومي عند التبديل

4. ✅ **معالج تغيير المستوى الوظيفي**
   - عند تغيير المستوى الوظيفي: تحديث السعر اليومي تلقائياً
   - إعادة حساب المجموع الكلي تلقائياً

---

### 🚀 كيفية الاستخدام:

#### **للمحافظات والمدن:**
1. اختر موظف
2. اختر مدينة من القائمة (مثل بغداد - الكرخ)
3. ⚠️ سيظهر حقل "العنوان الوظيفي" (اتركه كما هو)
4. السعر سيتم تعيينه من سعر المدينة تلقائياً

#### **للإيفادات خارج البلد:**
1. اختر موظف
2. اختر **خارج القطر/X** من القائمة
3. ✨ **سيختفي** حقل "العنوان الوظيفي"
4. ✨ **سيظهر** حقل "المستوى الوظيفي" بدلاً منه
5. اختر المستوى الوظيفي (منتسب، مسؤول وجبة، إلخ)
6. ✅ السعر سيتحدث تلقائياً حسب الجدول الصحيح

### 📝 جدول الأسعار (مثال):

**خارج القطر/1:**
- منتسب: 30,000
- مسؤول وجبة: 35,000 ✨
- مسؤول وحدة: 35,000
- معاون: 45,000
- رئيس: 55,000

**خارج القطر/2:**
- منتسب: 30,000
- مسؤول وجبة: 35,000
- مسؤول وحدة: 55,000 ✨
- معاون: 65,000

### 🧪 إذا أردت التحقق من نجاح الإصلاح:

اضغط **F12** في المتصفح → **Console** → انسخ والصق:

```javascript
console.log('🔍 test mission rates:');
const test1 = window.missionRates.find(m => m.name === 'خارج القطر/1' && m.responsibility_level === 'مسؤول وجبة');
console.log('خارج القطر/1 + مسؤول وجبة =', test1?.daily_rate, '✅ يجب أن يكون 35000');

const test2 = window.missionRates.find(m => m.name === 'خارج القطر/2' && m.responsibility_level === 'مسؤول وحدة');
console.log('خارج القطر/2 + مسؤول وحدة =', test2?.daily_rate, '✅ يجب أن يكون 55000');
```

**النتيجة المتوقعة:**
```
خارج القطر/1 + مسؤول وجبة = 35000 ✅
خارج القطر/2 + مسؤول وحدة = 55000 ✅
```
