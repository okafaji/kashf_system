## 🔧 التعديلات التفصيلية

### 1️⃣ تعديل دالة `calculateRow()` - المبيت الثابت

**الملف:** `public/js/payroll_manager.js`  
**السطور المتأثرة:** 705-720  
**النوع:** تعديل منطق الحساب  

#### ما تم تغييره:
- **القديم:** التحقق من إدخال المستخدم للمبيت والتحقق من أنه > 0
- **الجديد:** تعيين المبيت تلقائياً إلى 10,000 للمدن العادية و 0 للخارج

#### الكود القديم:
```javascript
// التحقق من المبيت - يجب أن يدخل المستخدم قيمة
if (!isOutsideCountry && days > 1) {
    const accFeeInput = $row.find('.js-acc-fee');
    if (accFeeInput.val() <= 0) {
        accFeeInput.addClass('border-red-500 bg-red-50');
        accAlert.removeClass('hidden');
        isValid = false;
    }
}
```

#### الكود الجديد:
```javascript
// المبيت ثابت 10,000 لكل ليلة (فقط للمدن العادية)
// خارج القطر بدون مبيت
const FIXED_ACCOMMODATION = 10000;

if (isOutsideCountry) {
    // خارج القطر: بدون مبيت
    accFeeInput.val(0);
    accAlert.addClass('hidden');
    accFeeInput.removeClass('border-red-500 bg-red-50');
} else {
    // مدينة عادية: مبيت ثابت 10,000
    accFeeInput.val(FIXED_ACCOMMODATION);
    accAlert.addClass('hidden');
    accFeeInput.removeClass('border-red-500 bg-red-50');
}
```

#### التأثير:
✅ المبيت يُعيّن تلقائياً بدون إدخال من المستخدم  
✅ إلغاء التحقق من الأخطاء لحقل المبيت  
✅ ضمان قيمة ثابتة تماماً
