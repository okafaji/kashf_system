# 🤖 النسخ الاحتياطي التلقائي

## 📋 الوصف

نظام نسخ احتياطي تلقائي لقاعدة البيانات يعمل بشكل ذكي عند إضافة كشوفات جديدة.

---

## ✨ كيف يعمل؟

### 📌 **متى يتم النسخ التلقائي؟**

عند **إدخال أول كشف في اليوم الجديد**:
- ✅ عند إنشاء كشف واحد يدوياً
- ✅ عند استيراد كشوفات من Excel

### 🎯 **الآلية:**

1. **التحقق من النسخة اليومية:**
   - يتحقق النظام: هل تم إنشاء نسخة احتياطية اليوم؟
   - إذا **نعم** → لا يفعل شيء
   - إذا **لا** → ينشئ نسخة احتياطية تلقائياً

2. **إنشاء النسخة:**
   - يأخذ نسخة من قاعدة البيانات فقط (سريع!)
   - يحفظها في `storage/auto_backups/`
   - يسجل في cache أنه تم إنشاء نسخة لهذا اليوم

3. **التنظيف التلقائي:**
   - يحذف النسخ الأقدم من **15 يوم** تلقائياً
   - يحدث عند كل نسخ احتياطي جديد

## ⚙️ التفاصيل التقنية

### 1️⃣ **الدالة الرئيسية:**
```php
BackupController::createAutomaticBackup()
```

**الاستدعاء:**
- في `PayrollController::store()` - عند إنشاء كشف واحد
- في `PayrollController::store_multiple()` - عند استيراد Excel

### 2️⃣ **آلية منع التكرار:**
```php
$cacheKey = "auto_backup_" . now()->format('Y-m-d');
if (Cache::has($cacheKey)) {
    return; // تم إنشاء نسخة اليوم
}
Cache::put($cacheKey, true, now()->endOfDay());
```

**الفوائد:**
- ✅ نسخة واحدة فقط في اليوم (لا تكرار)
- ✅ سريع (يتحقق من cache)
- ✅ ينتهي تلقائياً في نهاية اليوم

### 3️⃣ **التنظيف التلقائي:**
```php
private static function cleanOldAutoBackups()
{
    $cutoffDate = now()->subDays(15);
    // حذف المجلدات الأقدم من 15 يوم
}
```

**يحدث:**
- عند كل نسخ احتياطي جديد
- يحذف النسخ الأقدم من 15 يوم فقط
- يسجل في logs قائمة بالنسخ المحذوفة

## 📈 الأداء

### ⚡ **السرعة:**
| العملية | الوقت |
|---------|-------|
| إنشاء نسخة DB (~5-20 MB) | 1-3 ثواني |
| التحقق من Cache | 0.001 ثانية |
| حذف نسخة قديمة | 0.5 ثانية |

**التأثير على المستخدم:** صفر! لا يشعر بأي تأخير ⚡

### 💾 **المساحة:**
```
حجم النسخة الواحدة: ~5-20 MB
عدد النسخ المحفوظة: 15 نسخة (كحد أقصى)
المساحة الكلية: ~75-300 MB
```

## 🧪 الاختبار

### اختبار النسخ التلقائي:

1. احذف cache:
   ```bash
   php artisan cache:clear
   ```

2. أضف كشف جديد

3. تحقق من المجلد:
   ```bash
   Get-ChildItem storage\auto_backups
   ```

4. تحقق من الـ logs:
   ```bash
   Get-Content storage\logs\laravel.log | Select-String "Automatic backup"
   ```

### اختبار الحذف التلقائي:

1. أنشئ نسخة قديمة يدوياً (غيّر التاريخ في اسم المجلد)
2. أضف كشف جديد
3. تحقق من حذف المجلد القديم

---

## 📞 استكشاف الأخطاء

### المشكلة: "لا يتم إنشاء نسخ تلقائية"

**الحلول:**
1. تحقق من الصلاحيات:
   ```bash
   icacls storage\auto_backups /grant Everyone:F /T
   ```

2. تحقق من cache:
   ```bash
   php artisan cache:clear
   ```

3. راجع logs:
   ```bash
   Get-Content storage\logs\laravel.log | Select-String "backup"
   ```

### المشكلة: "النسخ لا تُحذف بعد 15 يوم"

**الحلول:**
1. تحقق من تنسيق اسم المجلد (يجب أن يكون: `auto_backup_YYYY_MM_DD_HH_MM_SS`)
2. راجع logs للتأكد من تشغيل cleanOldAutoBackups()
