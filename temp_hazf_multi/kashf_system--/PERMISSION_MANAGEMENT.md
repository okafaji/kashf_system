# 🔐 إدارة الصلاحيات - دليل الاستخدام

## 📌 النظرة العامة

تم إضافة نظام كامل لإدارة صلاحيات المستخدمين يسمح للمسؤولين بـ:
- عرض جميع المستخدمين وصلاحياتهم الحالية
- تعديل الصلاحيات والأدوار لكل مستخدم
- إضافة أو إزالة صلاحيات فردية بسهولة

## 🚀 كيفية الوصول

### الرابط المباشر:
```
http://your-domain/admin/permissions
```

### المتطلبات:
- يجب أن تكون مسجلاً دخول
- يجب أن يكون لديك أحد هذه الصلاحيات:
  - `manage-users` ✅
  - `manage-settings` ✅
  - دور `admin` ✅

## 📖 الميزات الرئيسية

### 1️⃣ عرض قائمة المستخدمين
الذهاب إلى `/admin/permissions` يعرض:
- جدول بجميع المستخدمين
- الأدوار الموكلة لكل مستخدم (مع شارة 🔵)
- الصلاحيات المباشرة (مع شارة ✓)
- زر التعديل لكل مستخدم

### 2️⃣ تعديل صلاحيات المستخدم
عند الضغط على "تعديل" لأي مستخدم:
- يتم عرض جميع الأدوار المتاحة مع checkbox
- يتم عرض جميع الصلاحيات مع checkbox (مصنفة حسب النوع)
- يتم عرض الحالة الحالية للمستخدم
- يمكن حفظ التغييرات بضغطة زر

### 3️⃣ العمليات المتاحة

#### شريط الأدوار:
```
☐ admin        ☐ user        ☐ manager
```
- اختر أحد أو أكثر من الأدوار

#### شريط الصلاحيات (مصنفة حسب النوع):
```
📊 عرض البيانات:
  ☐ access-dashboard
  ☐ view-payrolls
  ☐ view-employees

✏️ التعديل:
  ☐ create-payrolls
  ☐ edit-payrolls
  ☐ delete-payrolls
  
⚙️ الإدارة:
  ☐ manage-users
  ☐ manage-settings
```

**اختر ما تحتاج وانقر "حفظ"**

## 🔗 الروتس المسجلة

| الطريقة | الرابط | الاسم |
|--------|---------|-------|
| GET | `/admin/permissions` | عرض قائمة المستخدمين |
| GET | `/admin/permissions/{user}` | عرض تعديل صلاحيات مستخدم |
| PUT | `/admin/permissions/{user}` | حفظ التغييرات |
| POST | `/admin/permissions/{user}/grant` | إعطاء صلاحية واحدة (AJAX) |
| POST | `/admin/permissions/{user}/revoke` | إزالة صلاحية واحدة (AJAX) |
| POST | `/admin/permissions/{user}/assign-role` | إعطاء دور (AJAX) |
| POST | `/admin/permissions/{user}/remove-role` | إزالة دور (AJAX) |

## 💡 أمثلة الاستخدام

### مثال 1: إعطاء صلاحية Edit لمستخدم
1. اذهب إلى `/admin/permissions`
2. ابحث عن المستخدم واضغط تعديل
3. ضع علامة على `edit-payrolls` ✓
4. اضغط حفظ
5. سيرى المستخدم الآن زر التعديل على الكشوفات

### مثال 2: إعطاء دور Admin كامل
1. اذهب إلى `/admin/permissions`
2. ابحث عن المستخدم واضغط تعديل
3. ضع علامة على `admin` تحت الأدوار
4. اضغط حفظ
5. سيحصل المستخدم على جميع صلاحيات Admin

### مثال 3: إزالة صلاحية معينة
1. اذهب إلى edit صفحة المستخدم
2. أزل علامة من الصلاحية
3. اضغط حفظ

إذا حدثت مشكلة:
1. تأكد من تسجيل الدخول
2. تأكد من أن لديك الصلاحيات المطلوبة
3. جرب حذف cache: `php artisan cache:clear`
4. افحص `storage/logs/laravel.log` للأخطاء
