## 🔧 إذا واجهت مشكلة:

### **المشكلة: السعر لا يتحدث**

**الحل:**
1. افتح Console (F12)
2. اكتب: `console.log(window.missionRates.length)`
   - إذا < 40: البيانات لم تُحمل
3. اكتب: `$('.js-responsibility-level').length`
   - إذا = 0: لا توجد صفوف

### **المشكلة: حقل المستوى الوظيفي لا يظهر**

**الحل:**
1. اختر موظف
2. اختر **خارج القطر/X**
3. افتح Console وانظر للأخطاء

### **المشكلة: خطأ في Console**

**أرسل لي:**
```javascript
// انسخ هذا وأرسل النتيجة:

console.log($('.js-responsibility-level:visible').length);
```
