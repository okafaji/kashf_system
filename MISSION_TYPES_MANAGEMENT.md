# 📋 دليل إدارة أنواع الإيفادات (خارج البلد والمحافظات)

## 🎯 الوصول للصفحة

### من القائمة الجانبية:
```
الإعدادات → إدارة الإيفادات  [🎯 رابط برتقالي]
```

### أو عبر الرابط المباشر:
```
https://your-domain.com/mission-types
```

### الصلاحيات المطلوبة:
- **Permission**: `manage-settings`
- **Roles**: `admin` فقط (أو من لديهم صلاحية manage-settings)

---

## 📊 الصفحات الرئيسية

### 1️⃣ صفحة العرض والفلترة (`/mission-types`)

**الميزات:**
- ✅ عرض جميع أنواع الإيفادات الموجودة
- 🔍 البحث حسب:
  - نوع الإيفاد (خارج القطر/1، خارج القطر/2، إلخ)
  - مستوى المسؤولية (منتسب، رئيس، أمين عام، إلخ)
- ✎ تعديل كل سجل
- 🗑️ حذف السجلات غير المستخدمة
- 📊 عرض الإحصائيات

**الجدول يعرض:**
| العنصر | الوصف |
|------|------|
| نوع الإيفاد | مثال: خارج القطر/1 |
| مستوى المسؤولية | مثال: منتسب، رئيس، إلخ |
| المبلغ اليومي | السعر المستخدم في الحسابات (دينار) |
| الإجراءات | تعديل (✎) أو حذف (🗑️) |

---

### 2️⃣ صفحة الإضافة (`/mission-types/create`)

**الخطوات:**
1. اضغط على الزر **"➕ إضافة جديد"** من الأعلى
2. ملء البيانات:
   - **نوع الإيفاد**: اختر من القائمة أو أدخل نوع جديد (مثال: `خارج القطر/5`)
   - **مستوى المسؤولية**: اختر من القائمة أو أدخل مستوى جديد (مثال: `متدرب`)
   - **المبلغ اليومي**: أدخل السعر بالدينار (مثال: `25000.00`)
3. اضغط **"✓ حفظ"**

**ملاحظات:**
- ✓ لا يمكن إضافة نفس المجموع (نوع + مستوى) مرتين
- ✓ المبلغ يجب أن يكون رقم موجب
- ✓ يمكن استخدام أرقام عشرية (مثال: `30000.50`)

---

### 3️⃣ صفحة التعديل (`/mission-types/{id}/edit`)

**الطريقة:**
1. من جدول العرض، اضغط على أيقونة ✎ (تعديل) بجانب السجل المطلوب
2. عدّل القيم المطلوبة
3. اضغط **"✓ حفظ التغييرات"**

**يمكن تعديل:**
- ✏️ نوع الإيفاد (خارج القطر/1 → خارج القطر/2)
- ✏️ مستوى المسؤولية (منتسب → متدرب)
- ✏️ المبلغ اليومي (30000 → 32000)

**التحديثات ستؤثر على:**
- ✓ جميع الكشوفات الجديدة التي ستستخدم هذا النوع
- ✗ الكشوفات المحفوظة بالفعل (لن تتغير)

---

### 4️⃣ صفحة الإحصائيات (`/mission-types/stats`)

**المعلومات المعروضة:**
- 📊 إجمالي الأنواع المسجلة
- ✓ الأنواع المستخدمة فعلياً
- ✗ الأنواع غير المستخدمة
- 💰 إجمالي المبالغ المستخدمة

**الفائدة:**
- معرفة أي الأنواع لا تُستخدم (يمكن حذفها)
- تتبع الإيفادات المسجلة لكل نوع
- إجمالي المبالغ المصروفة حسب النوع

---

## 🔒 حذف السجلات

### متى يمكن الحذف؟
```
✓ يمكن الحذف إذا:
  - لم يتم استخدام النوع في أي كشف

✗ لا يمكن الحذف إذا:
  - موجود في كشف واحد أو أكثر
  - سيظهر رسالة خطأ: "لا يمكن حذف هذا النوع - مستخدم في X كشف"
```

### خطوات الحذف:
1. من جدول العرض، اضغط على أيقونة 🗑️ (حذف)
2. أكّد الحذف من رسالة التأكيد
3. تم حذف السجل بنجاح

## 🛠️ حالات الاستخدام الشائعة

### الحالة 1️⃣: تغيير سعر الإيفاد لمستوى معين
```
مثال: رفع سعر "منتسب في خارج القطر/1" من 30,000 إلى 32,000

الحل:
1. اذهب لـ "إدارة الإيفادات"
2. ابحث عن: نوع = "خارج القطر/1" & مستوى = "منتسب"
3. اضغط تعديل (✎)
4. غيّر المبلغ من 30,000 إلى 32,000
5. احفظ التغييرات
```

### الحالة 2️⃣: إضافة نوع إيفاد جديد
```
مثال: إضافة "خارج القطر/5" مع جميع المستويات

الحل:
1. اضغط "➕ إضافة جديد"
2. أدخل: نوع = "خارج القطر/5"، مستوى = "منتسب"، سعر = 30000
3. كرر للمستويات الأخرى

ملاحظة: يجب إضافة سجل منفصل لكل (نوع + مستوى)
```

### الحالة 3️⃣: إضافة مستوى مسؤولية جديد
```
مثال: إضافة مستوى "متدرب" لجميع الأنواع

الحل:
1. لكل نوع إيفاد، اضغط "➕ إضافة جديد"
2. أدخل: نوع = "خارج القطر/1"، مستوى = "متدرب"، سعر = 25000
3. كرر للأنواع الأخرى (2، 3، 4) بنفس المستوى
```

## ⚠️ تنبيهات مهمة

### ⚠️ التحديثات لا تؤثر بأثر رجعي
```
إذا قمت بـ:
1. إضافة كشف بـ "خارج القطر/1 - منتسب - 30,000"
2. ثم غيّرت السعر إلى 32,000
3. الكشف القديم سيبقى 30,000 (لن يتغير)
4. فقط الكشوفات الجديدة ستستخدم 32,000
```

### ⚠️ الحذف نهائي
```
عند حذف نوع إيفاد:
- لا يمكن استرجاعه
- تأكد من عدم استخدامه في الكشوفات الجديدة
```

## 📞 الدعم والمساعدة

### أسئلة شائعة:

**س: كيف أعرف إذا كان النوع مستخدماً؟**
- ج: اذهب للإحصائيات (📊) وابحث عن عدد الاستخدام

**س: هل يمكن تعديل نوع إيفاد مستخدم؟**
- ج: نعم، يمكنك تعديل المبلغ أو اسم المستوى في أي وقت

**س: ماذا لو حاولت حذف نوع مستخدم؟**
- ج: سيظهر خطأ يخبرك بعدد الكشوفات التي تستخدمها

## 🔄 التكامل مع باقي النظام

### أين تُستخدم هذه البيانات؟

1. **صفحة إنشاء الكشف** (`/payrolls/create`)
   - الأسعار تظهر تلقائياً عند اختيار "خارج القطر"
   - المستويات تظهر في dropdown "مستوى المسؤولية"

2. **صفحة تعديل الكشف**
   - يمكن تعديل المستوى والسعر من الكشف نفسه

3. **الحسابات**
   - اليومية = عدد الأيام × السعر من هذا الجدول
   - بدون مبيت (المبيت = 0)
