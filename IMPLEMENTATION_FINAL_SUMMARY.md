## 🎓 المفاهيم الأساسية

### Role (الدور)
```
مجموعة من الصلاحيات
مثال: دور "admin" = كل الصلاحيات
```

### Permission (الصلاحية)
```
قدرة محددة جداً
مثال: صلاحية "edit-payrolls" = تعديل الكشوفات فقط
```

### مثال عملي
```
User "أحمد":
  - يملك دور "admin"
  - والذي يشمل الصلاحيات: edit-payrolls, delete-payrolls, etc
  
User "محمد":
  - له صلاحية مباشرة: view-payrolls
  - بدون صلاحيات أخرى
```

## 🚨 معالجة الأخطاء الشائعة

### ❌ "خطأ 403 غير مصرح"
**الحل**:
```
تأكد من:
1. مسجل دخول ✓
2. لديك صلاحية manage-users أو manage-settings ✓
3. URL صحيح: /admin/permissions ✓
```

### ❌ "لا تظهر الأزرار للمستخدم"
**الحل**:
```
1. امسح cache: php artisan cache:clear
2. سجل دخول من جديد
3. تحديث الصفحة (F5)
```

### ❌ "الصفحة بيضاء / خطأ"
**الحل**:
```
شوف: storage/logs/laravel.log
ربما مفقود: PermissionController or views
```
